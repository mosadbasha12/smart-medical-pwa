# Smart Medical Installable PWA

هذه نسخة **تطبيق قابلة للتثبيت** على **Android** و **iPhone** من خلال المتصفح، مبنية على نفس فكرة مشروع التخرج:
- حجز المواعيد الطبية
- إدارة الملف الطبي
- متابعة الزيارات
- Chatbot للاستشارة الأولية

## ما الذي يميز هذه النسخة؟
- تعمل كتطبيق Web App قابل للتثبيت
- يمكن تشغيلها على **Android** و **iPhone**
- تحتوي على **وضع عرض Demo** حتى لو لم يكن السيرفر شغالًا
- تدعم الاتصال الحقيقي بالـ Backend إذا وضعت رابط الـ API
- تحفظ الجلسة محليًا وتعمل كواجهة موبايل كاملة

## التثبيت على Android
1. ارفع ملفات هذا المجلد على أي استضافة **HTTPS**.
2. افتح الرابط من **Chrome** على الموبايل.
3. من خيارات Chrome اختَر **Add to home screen / Install**.
4. سيظهر التطبيق كأنه تطبيق عادي على الشاشة الرئيسية.

## التثبيت على iPhone
1. ارفع ملفات هذا المجلد على أي استضافة **HTTPS**.
2. افتح الرابط من **Safari**.
3. اضغط **Share** ثم **Add to Home Screen** ثم فعّل **Open as Web App** إذا ظهر.
4. سيظهر التطبيق على الشاشة الرئيسية ويعمل كتطبيق مستقل.

## التشغيل المحلي للمراجعة
يمكنك تشغيله محليًا على الكمبيوتر:

```bash
python -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080/
```

## ربط التطبيق بالـ Backend
من داخل التطبيق:
- ادخل إلى **الحساب > إعدادات السيرفر**
- ضَع رابط السيرفر مثل:

```text
http://192.168.1.10:8000/api
```

إذا أدخلت الرابط بدون `/api` فالتطبيق سيضيفه تلقائيًا.

## ملاحظات مهمة
- وضع Demo يمنع الحجز المكرر لنفس الموعد داخل البيانات المحلية.
- النسخة هنا صالحة للعروض والمراجعة والتجربة السريعة.
- عند توفر Backend فعلي، التطبيق يتصل مباشرة بنقاط الـ API المعرفة في المشروع.
